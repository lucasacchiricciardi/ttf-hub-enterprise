# TTF Enterprise Hub

## Descrizione

TTF Enterprise Hub è il portale dedicato all'offerta formativa enterprise di Fondazione ITS TechTalentFactory. Questo progetto rappresenta un'iniziativa strategica di upskilling B2B che trasforma il know-how formativo della Fondazione in impatto concreto per le imprese italiane.

Il sito web presenta un ecosistema di soluzioni formative modulari (MVO - Minimum Viable Offers) organizzate per industry specifiche: System Integration & Software, Servizi & BPO, Pubblica Amministrazione & Sanità, e Manifatturiero & Industria 4.0.

## Caratteristiche Principali

- **Framework I.C.E.**: Approccio basato su Impact (ROI), Confidence (esperienza tecnica), e Ease (fattibilità immediata)
- **Reverse Engineering**: Metodo che decostruisce la formazione tradizionale per ricostruirla partendo dalle esigenze reali delle imprese
- **Catalogo Modulare**: Offerte formative iper-modulari da 40-80 ore per risolvere colli di bottiglia specifici
- **Industry Solutions**: Cataloghi dedicati per ICT, Servizi, PA, e Industria
- **Ecosistema Partner**: Oltre 100 aziende partner con placement rate del 90%

## Tecnologie Utilizzate

- **HTML5**: Struttura semantica delle pagine
- **Tailwind CSS**: Framework CSS utility-first per lo styling
- **Lucide Icons**: Libreria di icone vettoriali
- **Google Fonts (Inter)**: Font moderno per l'interfaccia
- **JavaScript Vanilla**: Interattività minima per icone e comportamenti base

## Installazione e Uso

Questo è un sito web statico che non richiede installazione complessa.

### Opzioni di Deployment:

1. **Locale**: Aprire `index.html` direttamente in un browser web
2. **Server Web**: Copiare i file su qualsiasi server web (Apache, Nginx, etc.)
3. **GitHub Pages**: Caricare il repository su GitHub e abilitare Pages
4. **Vercel/Netlify**: Deploy automatico tramite integrazione Git

### Navigazione:
- Dalla homepage, esplorare i diversi settori industry
- Ogni sezione contiene moduli formativi specifici per il settore
- Design responsive ottimizzato per desktop e mobile
